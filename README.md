# MovieProject
Шутеєв Дмитро
КН 2022-1
Для виконання курсового завдання я обрав мобільний застосунок MovieApp, оскільки він повністю відповідає технічним вимогам, зазначеним у методичних рекомендаціях. Проєкт реалізований мовою Kotlin, що є обов’язковою вимогою, та побудований за архітектурним підходом Single Activity, де основна логіка навігації винесена у фрагменти.
У застосунку використано Navigation Component: NavHostFragment, NavController, nav_graph.xml, а також інтегровано BottomNavigationView, завдяки чому навігація між екранами (Home та Favorites) виконується відповідно до вимог. Переходи між фрагментами реалізовані за допомогою Safe Args, що гарантує коректність передачі даних.
Для роботи з даними створено окремий клас MovieRepository, а для збереження обраних фільмів  менеджер FavoriteManager, що відповідає вимозі мати власну логіку обробки даних. Постери завантажуються за допомогою бібліотеки Glide, яка є стандартним інструментом для роботи із зображеннями у Android.
Отже, обраний мною проєкт повністю відповідає вимогам: він включає Kotlin, фрагменти, RecyclerView, навігацію, передачу аргументів, навігаційне меню, роботу з даними та використання зовнішньої бібліотеки. Проєкт демонструє коректну структуру Android-застосунку та охоплює всі необхідні концепції, що підтверджує його відповідність поставленим завданням.
